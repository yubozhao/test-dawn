Model

A model is referred to the product of machine learning experiment. They are function that when applied data, generate predicated values. Model can be deployed to one or more endpoints. Once deployed, models can be monitored for performance.

createModel

Parameters

Property NameFormatDescription
nameRequired - StringName of the endpoint
  • Name must be valid ASCII and may contain lowercase and uppercase letters, digits, underscores, and dashes.
  • Name may not start with a dash and may contain a maximum of 128 characters.
  • Name must be unique.
input_typeRequired - StringInput type for the endpoint
functionRequired - ?Function script
container_infoRequired - Objectcontainer info
container_info.registryRequired - Stringregister link
container_info.imageRequired - Stringimage name
container_info.tagRequired - Stringimage tag
versionOptional - StringEndpoint version. Default is latest.
labelsOptional - [Object]Labels for this endpoint
labels.keyRequired - StringLabel key
labels.valueRequired - StringLabel value

Response

  • id: [String] Model's ID after successful creation.
  • name: [String] Model's name
  • version: [String] Model's version

Example

CLI

$ atalaya create model -f data.json

Curl

curl -x POST -H 'Authorization: Bearer YOUR_ACCESS_TOKEN' \
  -d '{}' -H 'Content-Type: application/json' \
  https://api.atalaya.io/v1/models/create

GraphQL

input createModelInput {
  name: String!
  function: String!
  version: String
  labels: JSON
  container_info: JSON
  models: JSON
}

mutation (
  $data: createModelInput
) {
  createModel(
    input: $data
  ) {
    model {
      name
      version
      ...other fields you want to return
    }
  }
}

updateModel

nameRequired - StringName of the endpoint

functionOptional - ?Function script
container_infoOptional - Objectcontainer info
