Endpoint

Endpoint enables you deploy model as web services, manage various version of models, and monitor their performance and metrics.

createEndpoint

Create an endpoint that server prediction.

Parameters

Property NameFormatDescription
nameRequired - StringName of the endpoint
  • Name must be valid ASCII and may contain lowercase and uppercase letters, digits, underscores, and dashes.
  • Name may not start with a dash and may contain a maximum of 128 characters.
  • Name must be unique.
input_typeRequired - StringInput type for the endpoint
slo_metricRequired - NumberSLO(Service Level Object) metric, in millisecond.
modelsRequired - [Object]Models used for this endpoint
models.nameRequired - StringModel Name
models.verisonRequired - StringModel Version
versionOptional - StringEndpoint version. Default is latest.
labelsOptional - [Object]Labels for this endpoint
labels.keyRequired - StringLabel key
labels.valueRequired - StringLabel value

Response

  • name: [ String ] endpoint's name
  • version: [ String ] endpoint's version
  • url: [ String ] endpoint's prediction URL.

Example

CLI

$ atalaya create endpoint -f data.json

Curl

curl -x POST -H 'Authorization: Bearer YOUR_ACCESS_TOKEN' \
  -d '{}' -H 'Content-Type: application/json' \
  https://api.atalaya.io/v1/endpoints/create

GraphQL

input createEndpointInput {
  name: String!
  version: String
  labels: JSON
  slo_metric: Number
  models: JSON
  input_type: String
}

mutation (
  $data: createEndpointInput
) {
  createEndpoint(
    input: $data
  ) {
    endpoint {
      name
      version
      ...other fields you want to return
    }
  }
}

getEndpoint

Parameters

Property NameFormatDescription
nameRequired - StringName of the endpoint
versionOptional - StringEndpoint version. Default is latest.

Response

  • name: [ String ] endpoint's name
  • version: [ String ] endpoint's version
  • status: [ Object ]
    • is_deployed: *[ Boolean ] *
  • labels: [ [Object] ] Labels for this endpoint
    • key: [ String ] Label key
    • value: [ String ] Label value
  • input_type: [ String ] Input type for the endpoint
  • version: [ String ] Endpoint version. Default is latest.
  • slo_metric: [ Number ] SLO(Service Level Object) metric, in millisecond.
  • models: [ [Object] ] Models used for this endpoint
