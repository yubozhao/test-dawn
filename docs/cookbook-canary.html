Canary Deployment

What is Canary Deployment?

Canary deployment is when a new version of model arrives from training and the one currently serving traffic becomes the second-newest version, user can opt to serve both of those versions simultanously.

All predication traffic would still send to the second-newest version, while also send a sample of the traffic to the newest version to enable performance comparsion. Once there is enought confidence in the newest version, user would then switch to serving only the newest version.
